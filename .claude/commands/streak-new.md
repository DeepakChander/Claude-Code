# Create New Challenge

Initaite the "New Challenge" flow.

1.  **Initialize**: Ensure `.streak/` folder exists.
2.  **Ask Type**: Check `references/types.md` for supported types.
3.  **Setup**: Gather info (name, goal, cadence).
4.  **Create Files**: Generate `active.md` and the challenge folder based on `file-templates.md`.

**Instructions**:
Follow the creation flow defined in the `streak` skill logic (Flow 1: New Challenge).
Use `Write` tool to create the necessary files locally.
