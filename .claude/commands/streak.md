# Streak Check-in

Execute the standard check-in flow for the active challenge.

1. **Load Context**: Read `.streak/active.md` and the challenge configuration.
2. **Interact**: Ask user about their session (or plan upcoming session).
3. **Log**: Update `challenge-log.md` and `challenge-config.md`.

**Instructions**:
Follow the check-in flow defined in the `streak` skill logic (Flow 2: Check-in).
If no challenge is active, prompt to start a new one (Flow 1).
