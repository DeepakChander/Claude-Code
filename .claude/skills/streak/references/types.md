# Challenge Types Reference

Type-specific content for each challenge type: preferences sections, setup questions, ideation suggestions, and wrap-up questions.

---

## LEARNING Type
**Best for:** Courses, books, skills, certifications

### Setup Questions
- **What are you learning?** (topic/skill)
- **Any resources?** (courses, books, tutorials - optional)
- **How do you learn best?** (videos / reading / hands-on / mixed)

### Wrap-up Questions
- **Any aha moments?**
- **Progress on resources/milestones?**

---

## BUILDING Type
**Best for:** Projects, shipping, coding, apps

### Setup Questions
- **What are you building?** (project type or specific project)
- **Tech stack?** (languages, frameworks - optional)
- **Any linked projects?**

### Wrap-up Questions
- **What did you ship/complete?**
- **Any blockers?**

---

## FITNESS Type
**Best for:** Workouts, health goals, physical challenges

### Setup Questions
- **What's the goal?** (strength / cardio / weight loss / flexibility / general)
- **Equipment access?** (gym / home / bodyweight)
- **Preferred workout types?**

### Wrap-up Questions
- **What workout/exercises?**
- **How did your body feel?**

---

## CREATIVE Type
**Best for:** Art, writing, music, content creation

### Setup Questions
- **What medium?** (writing, art, music, video, etc.)
- **Any theme or focus?**
- **Tools you use?**

### Wrap-up Questions
- **What did you create?**
- **Any inspiration or themes?**

---

## HABIT Type
**Best for:** Routines, consistency, building discipline

### Setup Questions
- **What habit specifically?**
- **Trigger:** When/where/after what will you do it?
- **How long per session?**

### Wrap-up Questions
- **Did you complete the habit?**
- **How did it feel?**

---

## CUSTOM Type
**Best for:** Anything that doesn't fit other categories

### Setup Questions
- **What are you tracking?**
- **What fields do you want to track per session?**
- **What questions should I ask at each check-in?**
